# 🏛️ Hedera E-commerce Agent: Digital Heritage Edition

This is a Week 1 submission for the **Hedera AI Bounty**.

It's a fun, interactive AI checkout assistant built with the **Hedera Agent Kit** and **Gemini AI**. The agent sells legendary Web3 artifacts (like the 10,000 BTC Pizza and OG CryptoPunk) and executes real HBAR transactions on the testnet.

## ✨ Key Features
* **Native Intent Parsing:** Uses Google Gemini to understand user chat and buying intents.
* **Hedera Agent Kit Integration:** Built to securely interact with the Hedera network.
* **Human-in-the-Loop Security:** Features a strict "Yes/No" UI confirmation flow before any funds are transferred.
* **Instant Verification:** Instantly returns a live HashScan link upon successful purchase.

## 🛠️ Tech Stack
* Next.js (App Router) / TypeScript
* Hedera Agent Kit V4 & `@hashgraph/sdk`
* Google Gemini API
* Tailwind CSS
